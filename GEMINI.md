# Workspace Overview - Gemini Context

This repository (`Ronny25.github.io`) is a collection of web development projects, ranging from basic HTML/CSS tasks to more complex React applications. It serves as a personal portfolio and a record of technical progress.

## Repository Structure

The repository is organized into several sub-directories, each representing a different stage of learning or a specific project:

- **`MovieApp/`**: A legacy React (v15) / Redux application for browsing movies via TMDB API. (See `MovieApp/GEMINI.md` for details).
- **`jsX-Y/`**: JavaScript-focused exercises and small projects (e.g., `js11-12` contains carousel and fancybox plugins).
- **`taskX-Y/`**: HTML/CSS layout tasks and basic web components.
- **`finalExam/`**: A project likely serving as a course capstone, using Gulp for its build process.
- **`GLJSBasecamp/`**: Projects related to a specific JavaScript bootcamp or course.
- **`react-lessons/`**: Early React learning materials.

## Technical Context

- **Evolution:** The codebase shows an evolution from basic frontend tasks (HTML/CSS) to jQuery-based interactions (`js11-12`) and eventually to modern (for 2017) frameworks like React and Redux (`MovieApp`).
- **Tooling:** Different projects use different build tools including Gulp (`finalExam`, `js19-20`), Grunt (`js17-18`), and Webpack (`MovieApp`).
- **Standards:** Most legacy projects use older versions of libraries and older JS standards (ES5/ES6).

## Gemini Interaction Guidelines

- **Surgical Edits:** When working within a sub-project, strictly adhere to the local conventions and tooling of that specific directory.
- **Legacy Awareness:** Many projects are intentionally legacy. Do not suggest modernizations (like React Hooks or Vite) unless specifically asked.
- **Context Preservation:** Always refer to the specific `GEMINI.md` or `README.md` within a sub-directory before making changes.
